Functions

mt_addr:addSlots (slotList, count)
Adds a list of slots to the address. The slots added to the address will be removed from the provided slotList.

Parameters:

  • slotList a list of slots to be added
  • count the number of slots to take from the list provided, defaults to ALL if omitted

Returns:

    the address object
mt_addr:delete ()
deletes an address object. The address must have been disabled before. Deleting is automatically done after weight recalculation.

See also:

    disable
mt_addr:disable ()
disables an address object from the balancer. It will set its weight to 0 and the deleteMe flag to true. So the next slot-recalculation will delete the address.

See also:

    delete
mt_addr:dropSlots (slotList, count)
Drop an amount of slots and return them to the overall balancer.

Parameters:

  • slotList The list to add the dropped slots to
  • count (optional) The number of slots to drop, defaults to ALL if omitted

Returns:

    slotList with added to it the slots removed from this address
mt_balancer:addHost (hostname, port, weight)
Adds a host to the balancer.

Parameters:

  • hostname hostname to add
  • port (optional) the port to use (defaults to 80 if omitted)
  • weight (optional) relative weight (defaults to 10 if omitted)

Returns:

    balancer object, or throw an error if it already is in the list
mt_balancer:addWeight (delta)
Updates the total weight. Will mark the balancer as 'dirty' for recalculation of the slots

Parameters:

  • delta the in/decrease of the overall weight (negative for decrease)
mt_balancer:addressIter ()
Address iterator. Iterates over all addresses in the balancer (nested through the hosts)

Returns:

    weight (number), address (address object), host (host object the address belongs to)
mt_balancer:getPeer ()
Gets the next host according to the loadbalancing scheme.

Returns:

    hostname/ip and port
mt_balancer:redistributeSlots ()
Recalculates the weights. Updates the slot lists for all hostnames. Must be called whenever a weight might have changed; added/removed hosts.

Returns:

    balancer object
mt_balancer:removeHost (hostname)
Removes a host from a balancer. Will not throw an error if the hostname is not in the current list

Parameters:

  • hostname hostname to remove

Returns:

    balancer object

mt_host:queryDns ()

new (opts)

Creates a new balancer. The balancer is based on a wheel with slots. The slots will be randomly distributed over the hosts. The number of slots assigned will be relative to the weight.

The options table has the following fields;

  • hosts (required) containing hostnames and (optional) weights, must have at least one entry.
  • wheelsize (required) for total number of slots in the balancer
  • dns (required) a configured dns.client object for querying the dns server

Parameters:

  • opts table with options

Returns:

    new balancer object or nil+error
